Bienvenido a HTML

TML, o Lenguaje de Marcado de Hipertexto, es el estándar utilizado para crear páginas web. Permite estructurar el contenido mediante etiquetas, facilitando la creación de textos, imágenes, enlaces y otros elementos que forman parte de un sitio web.

¿Cómo hacer café?

  1. Muele el café justo antes de prepararlo.
  2. Hierve agua.
  3. Coloca un filtro.
  4. Agrega el café molido al filtro.
  5. Vierte el agua lentamente.
  6. Espera y deja que el café se infusione.
