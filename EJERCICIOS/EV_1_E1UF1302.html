¿Cómo se crea un documento HTML?

Aprender nunca había sido tan fácil!!!

Un documento HTML se crea utilizando etiquetas que definen la estructura y el contenido de la página. Comienza con la declaración <!DOCTYPE html>, que indica al navegador que el documento es HTML5. Luego, se utiliza la etiqueta <html> para envolver todo el contenido. Dentro de esta, se encuentran las etiquetas <head> para los metadatos y <body> para el contenido principal. Estas etiquetas trabajan juntas para estructurar y presentar la información en la web.

Estos son algunos conceptos relacionados con HTML:

  • Etiquetas HTML: Son los elementos que definen la estructura de un documento HTML. Por ejemplo, <h1> para encabezados, <p> para párrafos, y <a> para enlaces.
  • Atributos: Proporcionan información adicional sobre las etiquetas. Por ejemplo, el atributo href en un enlace define la URL a la que apunta.
  • Comentarios: Se utilizan para agregar notas dentro del código HTML y no se muestran en el navegador. Se escriben entre las etiquetas <!-- y -->.
  • Doctype: La declaración <!DOCTYPE html> indica al navegador que el documento es HTML5.
  • Encabezados: Se utilizan para definir títulos y subtítulos en la página. Hay seis niveles de encabezados, desde <h1> (el más importante) hasta <h6> (el menos importante).
  • Listas: Se utilizan para agrupar elementos relacionados. Hay listas ordenadas (<ol>) y no ordenadas (<ul>).
Etiqueta Descripción Ejemplo
<h1> Encabezado de nivel 1 <h1>Título principal</h1>
<p> Párrafo <p>Este es un párrafo.</p>
<a> Enlace <a href="https://www.ejemplo.com">Visitar ejemplo</a>
