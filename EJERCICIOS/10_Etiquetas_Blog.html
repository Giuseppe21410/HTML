¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Se utiliza para estructurar el contenido en la web y definir la presentación de los elementos en una página.

Se muestran algunas etiquetas fundamentales del lenguaje:

  • "<h1>" Abre un encabezado.
  • "<a>" Inserta un enlace.
  • "<p>" Abre un párrafo.

Listas en HTML

La diferencia entre las listas ordenadas y no ordenadas es que las listas ordenadas tienen un orden específico, mientras que las listas no ordenadas no tienen un orden específico.

Se ejemplifican y se muestran las características principales de cada una de las listas:

  1. Lista ordenada:
    • Evita la ambigüedad.
    • Se presenta ordenado.
    • Se utiliza para enumerar elementos.
  2. Lista no ordenada:
    • Se presenta desordenado.
    • Se utiliza para agrupar elementos.
    • Se utiliza para mostrar una lista de elementos sin un orden específico.

Estructura semántica.

La estructura semántica de un documento HTML es la forma en que se organizan y etiquetan los diferentes elementos del contenido para transmitir su significado y función. En HTML5, se introdujeron nuevas etiquetas semánticas que ayudan a definir la estructura del documento de manera más clara y comprensible.

Algunas de las etiquetas semánticas más comunes son:

  • <header>: Define el encabezado de un documento o sección.
  • <footer>: Define el pie de página de un documento o sección.
  • <main>:Agrupa el contenido principal de una página web.
